== Description ==

Gwangi is a Dating & Community theme that integrates seamlessly with BuddyPress and WooCommerce. It makes good use of the Customizer, allowing you to build a tailor-made community plateform using custom color, typography and layout options.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the 'Add New' button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
